<p align="center">

* Linux:
```
git config --global alias.gl 'log --graph --all --date=format-local:"%d-%b %H:%M" --format=format:"%C(bold blue)%h%C(bold yellow)%d %Creset| %C(bold green)%ad %Creset- %C(red)%an%Creset %n%w(80,4,4)%B"'
```

* Windows:
```
git config --global alias.gl 'log --graph --all --date=format-local:\"%d-%b %H:%M\" --format=format:\"%C(bold blue)%h%C(bold yellow)%d %Creset| %C(bold green)%ad %Creset- %C(red)%an%Creset %n%w(80,4,4)%B\"'
```
